Opis projekta

Movie Catalog je Android aplikacija razvijena u Java jeziku u Android Studio okruženju. Aplikacija omogućava pregled liste filmova, detaljan prikaz svakog filma, pregled glumaca, pretragu filmova, ocjenjivanje i dodavanje filmova u favorite.

Funkcionalnosti
prikaz liste filmova
detaljan prikaz filma
prikaz glumaca po filmu
pretraga filmova
ocjenjivanje filmova (rating bar)
dodavanje i prikaz favorita
navigacija između ekrana pomoću BottomNavigationView
Tehnologije
Java
Android Studio
RecyclerView
Adapter pattern
Intent (navigacija između aktivnosti)
Filterable (pretraga)
Struktura aplikacije
MainActivity – prikaz filmova
MovieDetailsActivity – detalji filma
FavoritesActivity – lista favorita
ActorsActivity – prikaz glumaca
MovieAdapter – prikaz filmova u listi
Napomena
Favoriti se čuvaju samo tokom rada aplikacije (bez baze podataka).

 Autor
